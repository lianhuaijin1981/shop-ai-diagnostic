# 线下门店AI经营诊断系统 (shop-ai-diagnostic) 开发进度

## 项目信息
- **仓库地址**: https://github.com/lianhuaijin1981/shop-ai-diagnostic
- **技术栈**: NestJS 10 + MongoDB + Redis | React 19 + TypeScript + Zustand + ECharts
- **部署方式**: Docker + CI/CD
